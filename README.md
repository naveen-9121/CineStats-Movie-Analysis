# CineStats-Movie-Analysis
Movie industry data analysis using MySQL — exploring box office trends,  genre popularity, director performance and revenue insights on a 500+ record IMDB-style dataset.


## 🎬 CineStats – Movie Industry Insights Engine

This project analyzes a movie dataset using SQL to uncover insights about revenue, popularity, and industry trends.

- Processed and analyzed 500+ movie records
- Used SQL queries (JOIN, GROUP BY, HAVING, ORDER BY)
- Calculated ROI and identified high-performing movies
- Analyzed genre popularity and director performance
- Explored seasonal trends in movie releases

This project demonstrates strong skills in SQL, data analysis, and database management.
